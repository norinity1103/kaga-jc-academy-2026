# 01. プロジェクトのコンテキスト ／ 上位方針との接続

## プロジェクト目的（再掲）

加賀LOMの既存会員のJC活動参加率を、**相互理解・帰属意識・支え合う文化の醸成**を通じて構造的に向上させる。  
単純な出席率向上ではなく、「挑戦する人を孤立させない」文化の確立をゴールとする。

## 加賀JC 2026年度 上位方針との接続マップ

議案上程時、説得力を最大化するために必ず参照する整合表。

| 加賀JC2026の文脈（理事長所信より） | 本議案との接続点 |
|---|---|
| 「不完全である勇気」—挑戦を通じて未来を切り拓く— | 挑戦する人を孤立させない文化＝この所信を体現する事業 |
| 入会3年未満が会員の**約7割** | 既存会員でも実質「育成期」が大半。アカデミー対象として正当性が高い |
| 「単なる人数の増加ではなく、自然に人が集まる魅力ある加賀JC」 | 参加率向上の本質的目的を明示 |
| 会員開発委員会 設置目的「リーダーとしての資質を磨き、地域変革の起点となる人財を育成」 | 本議案の上位委員会方針との一致 |
| 副理事長への期待「新入会員の成長支援や会員の定着」「一人ひとりがリーダーシップを発揮できる環境づくり」 | 副理事長案件として支持を得やすい |
| 専務理事への期待「無理なく関われる仕組みや環境」 | 「全員同じ熱量での参加は難しい」前提と完全合致 |
| 事務局「無理を強いる場ではなく、共に成長し共に調和を模索できる場」 | 議案背景にそのまま引用可能 |

**出典：加賀青年会議所 2026年度 理事長所信**  
https://www.kaga-jc.or.jp/rijityou/

## 中期ビジョン「シン・加賀ＪＣ構想」との接続

加賀JCは創立60周年を機に「シン・加賀ＪＣ構想 〜カガイッシンを創造せよ〜」を掲げている。3つのイッシンとの本議案の対応：

| イッシン | 内容 | 本議案との関係 |
|---|---|---|
| 一新 | 歴史と最先端技術・情報を融合し、革新的な加賀の発展を目指す | 既存会員の関わり方をアップデートする位置付け |
| 一進 | 境界を越え挑戦する人財を育成し、地域の未来を創造する | **本議案の中核**：挑戦する人材を育成・支援 |
| 一心 | すべての世代と地域が一つとなり、加賀への想いを深める | 既存会員同士の相互理解＝LOM内の一心 |

## 議案の位置付け

- 会員開発委員会の所管事業
- 単発の例会／例会アワーまたは複数回のアカデミー塾としての設計が可能
- 既存会員（特に入会3年未満〜中堅）を主対象としつつ、ベテランも巻き込む構成
- 加賀絆育成委員会／財務・例会委員会との連携余地あり
