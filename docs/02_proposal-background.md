# 02. 議案背景サマリ

## 1. 事業背景（議案「実施に至る背景」に投入する論旨）

現在の加賀LOMでは、JC活動への関わり方や主体性にメンバー間で差が生じており、積極的に行動する一部のメンバーに役割・責任・負担が集中しやすい状況がある。挑戦を続けるメンバーほど疲弊し、時に孤立してしまうケースが見受けられる。

加えて近年は、仕事・家庭・地域での役割などメンバーを取り巻く環境が多様化しており、全員が同じ時間・熱量で活動に参加することが難しくなっている。

このまま一部の積極的なメンバーだけが挑戦と責任を担い続ける状態が続けば、組織全体の主体性が低下し、受け身の姿勢が広がり、LOMとしての一体感が失われる恐れがある。

そのため本事業では、単に「参加率を上げる」ことを目的とするのではなく、既存会員同士の相互理解を深め、JC活動の目的や意義を再確認し、一人ひとりが仲間を支える意識を持つことを目指す。全員で支え合う文化を根付かせることで、挑戦する人を孤立させず、主体的にJC活動へ関わる人財を増やし、加賀LOM全体の参加意欲と一体感を高める。

## 2. 中心に置くべき課題（複合課題）

| 観点 | 課題 |
|---|---|
| 主体性 | 活動に積極的なメンバーと受け身のメンバーの差がある |
| 負担集中 | 行動する人に責任や役割が偏っている |
| 孤立 | 挑戦するメンバーが支えられず疲弊しやすい |
| 相互理解 | メンバー同士の事情・価値観・考え方を知る機会が不足 |
| 帰属意識 | 「自分も組織の一員である」という意識が弱まりやすい |
| 参加意欲 | 活動目的や役割が見えにくく、参加が自分ごとになりにくい |

## 3. 事業の論理構造（Logic Model）

```
メンバー同士を知る
   ↓
JC活動の目的・意義を理解する
   ↓
自分の役割を見出す
   ↓
仲間を支える意識を持つ
   ↓
挑戦する人が孤立しない文化をつくる
   ↓
結果として参加率・主体性・一体感が向上する
```

→ 参加率向上は**目的**であると同時に**結果指標**。手段としては「相互理解」「帰属意識」「支え合う文化の醸成」を重視。

## 4. 仮説（議案で証明すべきこと）

既存会員の参加率を高めるには、単に参加を呼びかけるだけでは不十分である。

参加率向上には、メンバー同士の**相互理解**、JC活動の**目的理解**、自分の**役割認識**、**仲間を支える意識**の醸成が必要である。

そのため、参加型コンテンツやグループワークを通じて、会員同士が関わり合いながら学ぶ機会を設けることで、組織への帰属意識と主体性が高まり、結果としてJC活動への参加意欲向上につながる。

→ この仮説は公開議案リサーチで十分に裏付けられている（詳細は `03_research-summary.md` および `04_case-references.md` 参照）。

## 5. 議案上での説得構造（5段論法）

1. **環境認識**：加賀JCの会員構成は入会3年未満が約7割。本業多忙化・家庭事情多様化が進む。
2. **課題定義**：そのため一部の積極会員に負担が集中し、孤立・疲弊・主体性低下の悪循環が起きている。
3. **打ち手の方向性**：相互理解と支え合う文化を醸成すれば、結果として参加率・主体性・一体感が向上する（仮説）。
4. **エビデンス**：小田原JC・大阪JC・ビジネスゲーム研究所事例で同様の仮説が実証されている。
5. **加賀LOMでの実装**：本事業として具体施策・KPI・スケジュールを設計する。
