## このプロジェクトを Claude Code で扱う際のコツ

- **長文の議案文を一発で書かせない**：背景／目的／効果／手法を別々に書かせて、後で統合する方が品質が高い
- **書式は触らない**：`reference/議案本文.htm` の CSS／テーブル構造を変えない指示を毎回明示する
- **加賀JC所信の文脈を必ず引かせる**：「加賀JC2026所信との整合を確認してから書いて」と毎回前置きする
- **数字を曖昧にさせない**：「向上を目指す」ではなく「+10pt」「+3名」と具体化を要求する
