# 03. 公開議案リサーチ サマリ

## 調査スコープ

JCの公開議案・公開所信・公開資料を横断調査し、加賀LOMアカデミー事業の議案構成への適用可能性を整理する。
gian-pocket.com 本体は会員ログイン必須のため、非ログインで閲覧可能な公開資料のみを対象とした。

## 横断的に見えた論点

公開議案を横断すると、加賀LOM議案で**必ず触れるべき定型論点**が明確になる。

| 議案の章 | 公開議案で頻出するキーワード |
|---|---|
| 背景（マクロ） | 平均年齢35歳化／入会3年未満が半数以上／会員数横ばい・減少／本業多忙化 |
| 背景（LOM固有） | 出席率の現状値／退会者数／拡大停滞要因 |
| 目的 | 当事者意識の醸成／仲間意識／自走するJAYCEE育成／好循環の構築 |
| 提供価値 | 共通価値観／役割理解／非日常体験／実践型学習／斜めの関係 |
| 手法 | 3レベル分割／模擬理事会／メンター制度／家族参加／オンライン併用／褒章 |
| KPI | 出席率%／拡大純増数／継続率／メンター稼働率／100%例会達成回数 |
| 上位整合 | ブロック方針・会頭所信・JCプログラム・JAYCEE育成カリキュラムとの紐付け |

## カテゴリ別 主要事例（要点のみ）

### A. 既存会員の参加率向上

- **小田原JC 創立70周年アクションプラン（5ヵ年）**：負の連鎖図（出席率↓→交流↓→拡大↓→退会↑→当事者意識↓）を明示し、5ヵ年で出席率を63%→80%に引き上げる数値計画。入会2年未満メンバーへのメンター制度導入。
- **日本JC 100%例会達成レポート**：全例会100%出席のLOMを表彰する全国共通指標。
- **日本JC 育LOM推進プロジェクト**：家庭と両立可能なLOM環境の認定制度。
- **東京JC 例会前後アンケート**：認知変化の効果測定の標準テンプレ。

### B. アカデミー × 既存会員の育成

- **大阪JC 会員開発委員会／資質向上委員会（2026年度室方針）**：「主体者意識と他者への思いやり」を5段階の体験ステップで育成する設計。
- **兵庫ブロック協議会 2026年度基本方針**：ブロックアカデミーでJAYCEE育成カリキュラムを運用、共通価値観学習で仲間意識を醸成。
- **神奈川ブロック協議会 2026年度方針**：「仲間の輝き増進委員会」が中核人財育成セミナーとかながわアカデミーの二層構造を採用。
- **東京BC／山梨BC 2026年度方針**：入会3年未満が半数以上という全国共通課題。

### C. 主体性・帰属意識の醸成

- **帯広JC アカデミースクール／模擬理事会**：10テーマから班ごとに選択→模擬議案構築→模擬理事会討議の実践型カリキュラム。
- **岡山ブロック協議会 ブロックアカデミー**：入会3年未満／役員候補生／正副・常任の**3レベル分割セミナー**。
- **埼玉ブロック協議会 アカデミー塾**：天光寺での修行体験（滝行）など非日常体験で「仲間との結束を深める」設計。

### D. グループワーク・参加型コンテンツの実証成果

- **ビジネスゲーム研究所のJC例会導入事例**：
  - 90分構成テンプレ（開会10分→ルール説明10分→ゲーム50分→振り返り15分→閉会5分）
  - 報告された成果：①新入会員1年以内離脱率ゼロを達成したLOM ②普段意見を控える会員がゲーム内でリーダー役→自信→委員会で立候補 ③異業種間の距離が縮まり事業提携に発展
  - 振り返りの問い：「JC活動でチャレンジしてみたいことは？」
- **JCプログラム（日本JC公式）**：Purpose、JCI Achieve、JCI Impact、VMV、JCゲーム等。公式コンテンツとして議案承認を得やすい。

## 仮説検証の結論

`02_proposal-background.md` §4 の仮説は公開議案レビューで**十分に裏付けられる**。

### 特に強い裏付け
- ビジネスゲーム研究所事例で「ゲーム内リーダー体験→自信→委員会で立候補」という主体性発火の具体例が報告されている
- 小田原JCは5年計画で「当事者意識の醸成」を出席率向上の起点に据えた
- 加賀JC2026理事長所信そのものが「共に成長し共に調和を模索できる場」を組織運営の根幹に置いている
- 大阪JC会員開発委員会が「主体者意識と他者への思いやりを兼ね備えたリーダー育成」を5段階の体験設計で実装している

### 注意すべき点
- 「グループワーク＝必ず効果が出る」は誤り。**振り返り設計の質**が成果を決める
- 既存会員向けは新人向けより**目的の言語化が難しい**。「なぜ今この事業が必要か」の説明力が議案の生命線
- 非日常体験は強力だが、家庭事情の多様化を踏まえた**参加ハードル設計**が必須

詳細な事例カードは `04_case-references.md` に集約。
