# 04. 事例カード集（議案ごとの整理）

公開議案・公開所信・公開資料を、議案構成へ流用できる粒度で整理したカード集。

## A-1. 小田原JC 創立70周年アクションプラン（5ヵ年公開PDF）

| 項目 | 内容 |
|---|---|
| 出典 | 公益社団法人小田原青年会議所 |
| URL | https://www.odawara-jc.or.jp/2025/action70/70plan.pdf |
| 事業目的 | 当事者意識の醸成→活気に満ちたLOMの実現（5年計画） |
| 背景課題 | ①出席率低下 ②交流機会減少 ③会員数減少 ④退会増加 → 当事者意識の低下（負の連鎖を図式化） |
| 実施内容 | 出席向上（子連れ可・モーニング例会）／交流増加（月1交流・家族例会・業種別会）／拡大（毎年純増10名）／退会減少（入会2年未満メンバーへのメンター制度・委員長会フォロー） |
| 加賀LOMに使える点 | 「負の連鎖図」をそのまま背景セクションの図解に流用可。出席率と仲間意識を**因果でつなぐ思考法**が秀逸。KPIを年度別に数字で切る方式 |
| 注意点 | 5年計画なので単年度議案にスケールダウンする際は「初年度の現状値把握」に絞ると良い |

| 事業目的 | 主体者意識と他者への思いやりを兼ね備えたリーダーの育成 |

| 注意点 | 大阪JCの規模感（数百名）が前提。加賀LOMの規模に圧縮する必要あり |

## D-1. ビジネスゲーム研究所 JC例会導入事例（公開コラム）

| 項目 | 内容 |
|---|---|
| 出典 | ビジネスゲーム研究所 |
| URL | https://business-games.jp/jc_businessgame5s/ |
| 事業目的 | マンネリ化・温度差・一部偏重・新人疎外感の解消 |
| 背景課題 | 「内容が堅苦しく参加意欲が上がらない」「一部メンバーに発言が偏る」 |
| 実施内容（90分構成テンプレ） | 開会10分→ルール説明10分→ゲーム50分→**振り返り15分**→閉会5分。代表例＝NASAゲーム／マシュマロチャレンジ／砂漠からの脱出／報連相パズル等 |
| 加賀LOMに使える点 | 「楽しさ」と「学び」を両立する具体的な時間配分テンプレ。振り返りの問い例：「JC活動でチャレンジしてみたいことは？」 |
| 報告された実証成果 | ①新入会員1年以内離脱率ゼロを達成したLOM ②普段意見を控える会員がゲーム内でリーダー役→自信→委員会で立候補 ③異業種間の距離が縮まり事業提携に発展 ④満足度アンケートで「JC活動意義を再確認」「次回例会参加率向上」 |
| 注意点 | ゲームをやって終わりにせず、**振り返り設計が成果を決める**。「気づき→次の行動」への接続が必須 |
