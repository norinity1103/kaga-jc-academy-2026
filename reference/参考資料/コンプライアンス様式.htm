Q1 受付表は全議案に必要です。ご用意できていますか？

Q2 著作物の使用がありますか？（複数回答可）
✅
A ポスター・チラシ・映像等を使用します → Q6へ カード制作（BASEYARZ株式会社委託）
○
B 講師・演者が出演します
○
C パワーポイント・配布資料を使用します（フリー素材のみ利用）
✅
D フリー素材・著作権法32条に基づく引用を行って著作物を使用します → 様式8提出 加賀市公式サイト フロアガイド画像・テキスト引用（出所表記あり）
○
E 音楽を使用します
○
F 著作権の保護を受ける他人の著作物を利用しない
Q3 各種承諾書の注意点を確認してください
✅
A [様式5] 映像・ポスター等作成依頼承諾書「kisoku」フォルダに格納し原本を1部提出 BASEYARZ株式会社 / カードデザイン・印刷委託 【1件】
○
B [様式6] 講師等出演依頼承諾書「kisoku」フォルダに格納し原本を1部提出
○
C [様式7] 著作物利用等に関する承諾書「kisoku」フォルダに格納し原本を1部提出
✅
D [様式8] 著作物の引用における出所表記の確認及び報告書「kisoku」フォルダに格納 加賀市公式サイト フロアガイド 引用【1件】

○
B こちらで素材を提供します
▌ パワーポイントや配布資料等の使用について
Q12 外部協力者の写真やプロフィール等の掲載、著作物の引用等はありますか？
○
A 写真・プロフィールの掲載があります
✅
B 情報・文章等の引用、転載があります → Q13へ 出所表記あり ／ 加賀市公式サイト フロアガイド画像・テキスト
○
C 掲載・引用等はありません
Q13 引用される側の著作物は公表されているものですか？
✅
A 公表されています → Q14へ 加賀市公式Webサイト — 一般公開済みコンテンツ
○
B 公表されていません（→ 引用不可）
Q14 引用される側の著作物は附従的なものですか？（引用が主、被引用が従の関係）
✅
A 附従的なものです → Q15へ フロアガイド画像・テキストは本文説明の補足として附従的に使用
○
B 附従的ではありません（→ 引用要件未充足の可能性あり）
Q15 引用の客観的必要性はありますか？
✅
A 客観的必要性があります → [様式8] を記載し、引用部分を明確に区分して出所を明示した上で引用 ✔ 著作権法第32条の要件を充足 — 引用適法と判断
○
B 客観的必要性がありません（→ 引用不可）
