# 加賀LOMアカデミー事業 議案構築プロジェクト

公益社団法人 加賀青年会議所 2026年度 会員開発委員会  
既存会員のJC活動参加率向上を目的としたアカデミー事業の議案構築プロジェクト。

---

## このプロジェクトについて

加賀LOMでは、JC活動への関わり方や主体性にメンバー間で差が生じており、積極的に行動する一部のメンバーに役割・責任・負担が集中しやすい状況がある。挑戦を続けるメンバーほど疲弊し、時に孤立してしまうケースも見受けられる。

本プロジェクトは、単に「参加率を上げる」のではなく、**相互理解・帰属意識・支え合う文化の醸成**を通じて、結果として参加率・主体性・一体感が高まる事業を議案として設計するための作業空間。

## クイックスタート（Claude Code 利用者向け）

1. プロジェクト直下で Claude Code を起動
   ```bash
   cd kaga-jc-academy
   claude
   ```
2. `CLAUDE.md` を読み込んでもらう（自動で読まれます）
3. 依頼例：
   - 「議案の背景文を300字に圧縮して」
   - 「事業内容の手法セクションを90分タイムテーブル付きで書いて」
   - 「KPIに合わせた事業後アンケートを設計して」
   - 「上位方針との整合を1ページに整理して」
